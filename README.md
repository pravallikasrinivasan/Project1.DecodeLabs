The Responsive Architecture
A modern, responsive web application that showcases architectural designs and layouts with an adaptive user interface across different devices.

Features
 Fully responsive design (mobile, tablet, desktop)
 Architecture showcase (layouts, structures, visuals)
 Smooth navigation and structured sections
 Clean and modern UI design
 Fast loading and optimized performance

Technologies Used
> HTML5
> CSS3 (Flexbox / Grid)
> JavaScript
